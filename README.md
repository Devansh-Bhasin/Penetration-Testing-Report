# Metasploitable2 Security Assessment

This repository contains a security assessment of Metasploitable2 conducted in an isolated VirtualBox lab environment.

## Overview

The purpose of this project was to practice ethical hacking methodology, network reconnaissance, service enumeration, vulnerability assessment, and professional security reporting.

All testing was performed in a controlled lab environment using intentionally vulnerable systems.

## Lab Environment

- Kali Linux - Assessment machine
- Metasploitable2 - Vulnerable target machine
- VirtualBox Host-Only Network
- Private lab subnet

## Tools Used

- Nmap
- Netcat
- Enum4linux
- Metasploit Framework
- Wget

## Skills Demonstrated

- Network reconnaissance
- Port scanning
- Service enumeration
- SMB enumeration
- Vulnerability identification
- Controlled exploitation validation
- Risk assessment
- Security documentation
- Remediation planning

## Report

## Key Findings

- Multiple outdated services exposed on the target system
- Weak service configurations
- SMB enumeration exposure
- Known vulnerable services present
- Critical-risk attack surface in the lab target

## Disclaimer

This project was completed for educational purposes only in a private, isolated lab environment. No real-world systems, public networks, or unauthorized targets were tested.
